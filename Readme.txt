Ghi chú:
+) Sử dụng phiên bản JDK 15.0.1
+) Các thư viện yêu cầu cài đặt thêm:
   - Thư viện javafx: javafx-sdk-11.0.2
   - Thư viện javafx sử dụng SceneBuider: javafx-fxml
   - Thư viện kết nối SQL: mssql-jdbc-9.2.1.jre15.jar
Lưu ý:
+) Nên sử dụng SQL Server 2019 để backup Database
để tránh bị lỗi khi backup đối với các phiên bản SQL cũ hơn.
+) Có thế backup Database bằng file .bak hoặc 
có thể import Database bằng file .bacpac
